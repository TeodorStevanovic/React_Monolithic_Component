## ArticlesManager Component

- This React component represents an application for managing a list of articles. It uses React’s useState hook to manage state and includes logic for adding, removing, and toggling the display of article details.

# Features

- Article List: Articles are stored in the articles state as an array of objects. Each article has an id, title, summary, and display (to control the visibility of the summary).
- Add Article: Users can add a new article by entering a title and summary. Each new article receives a unique ID generated by a generator function.
- Remove Article: Allows removing an article from the list by its id.
- Toggle Summary Display: Clicking an article toggles the visibility of its summary between none and block.

# Component Structure

# State:

- articles: array of articles.
- title: current input for the new article’s title.
- summary: current input for the new article’s summary.
- ID Generator: A generator function (id) that creates unique numbers for article IDs.

# Functions:

- clickAdd: adds a new article to the list and resets the input fields.
- removeArticle: removes an article by its ID.3
- toggleArticle: toggles the display state of an article’s summary.
- changeTitle and changeSummary: track changes in the input fields.

# Used Components

- AddArticle: component for entering a new article with title and summary fields.
- ArticlesList: component that displays the list of articles and handles user interaction.

# Notes

- This is a monolithic component containing all logic and state within a single file.
- The ID generator is implemented as a generator function to ensure unique identifiers.
- Summary display is controlled via inline CSS styles with display: none or display: block.
